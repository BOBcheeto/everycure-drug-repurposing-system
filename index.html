Drug Repurposing Research Tool

A tiny EveryCure‑style system

Old drugs, new possibilities. This system scans a small database of drugs, diseases, and biological targets — and flags cases where a drug already in use might treat a disease it was never prescribed for.

How the system works

The logic is simple: if a drug and a disease are both connected to the same biological target (a protein or enzyme in the body), there may be a medical reason to try that drug on that disease. The system finds these hidden connections automatically.

01

Store what we know about each drug

Each drug is listed alongside its approved uses (what doctors currently prescribe it for) and its biological targets — the proteins or enzymes inside the body that the drug interacts with.

02

Store what we know about each disease

Each disease is linked to a set of biological targets that scientists have found to be involved in causing or worsening it. For example, the protein NF-kB is linked to Cancer, Arthritis, Alzheimer's, and Heart Disease.

03

Find the overlaps

The matching engine compares every drug against every disease. When they share at least one biological target, that pair becomes a candidate. The system uses a Python set() intersection — the same idea as a Venn diagram.

04

Score and rank the matches

Each match gets a score equal to the number of shared targets. More overlap means stronger biological evidence. Results are sorted so the highest-scoring matches appear first.
